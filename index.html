🏗️ Burbopla Desktop v1.0

Burbopla Desktop es una aplicación de escritorio diseñada para optimizar la gestión de empresas de construcción. Con una interfaz intuitiva y herramientas eficientes, permite administrar empleados, proyectos, materiales y herramientas de manera centralizada.

Características principales:

  • ✔️ Gestión integral de empleados, proyectos, materiales y herramientas.
  • ✔️ Interfaz moderna y fácil de usar con JavaFX y Scene Builder.
  • ✔️ Base de datos MySQL para un almacenamiento seguro y estructurado.
  • ✔️ Instalador profesional con NSIS para una distribución sencilla.
  • ✔️ Flujo de trabajo optimizado para mejorar la eficiencia operativa.

Tecnologías utilizadas:


Burbopla Desktop es la solución ideal para empresas que buscan digitalizar y optimizar su gestión de proyectos de construcción. 🚀
